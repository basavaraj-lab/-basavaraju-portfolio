🐛 QUICK TROUBLESHOOTING:

Q: Port 8000 already in use?
A: Kill process: lsof -i :8000 | grep -v COMMAND | awk '{print $2}' | xargs kill -9

Q: Images not loading?
A: Check paths start with ../static/images/
   Verify files exist in backend/static/images/

Q: Styles not applying?
A: Clear browser cache (Ctrl+Shift+R)
   Verify CSS path in HTML: ../static/css/style.css

Q: Dependencies missing?
A: Run: cd backend && npm install
